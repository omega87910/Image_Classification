# Image_Classification
## Introduction 
手寫數字分類可以說是AI領域的"Hello World"，因此本區放上範例程式碼，供AI領域新手學習。
## Requirement
1. Anaconda  
2. tensorflow-gpu  
## Usage 
解壓縮<code>mnist_png.zip</code>  
啟動 Anaconda  
開啟 Jupyter notebook  
開啟本專案的<code>Mnist_model.ipynb</code>檔案  
## Reference
Mnist資料集來自 https://github.com/myleott/mnist_png <br>
